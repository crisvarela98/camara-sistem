# Sistema de monitoreo de camaras

Sistema simple de monitoreo de cámaras desarrollado en Python utilizando OpenCV.

## Funciones

- Captura de video en tiempo real
- Detección de movimiento
- Grabación automática de eventos
- Almacenamiento de clips con timestamp

## Tecnologías

Python  
OpenCV  
Computer Vision  

## Uso

Instalar dependencias:

pip install -r requirements.txt

Ejecutar:

python monitor.py

Presionar *Q* para salir.
